# Skills Demonstrated

This document maps the project files to the job-relevant skills demonstrated in the Global Store Retail Performance & Profitability Analysis project.

## Data Analyst and BI Analyst Skill Map

| Skill | Evidence in Repository | Why It Matters for Industry |
| --- | --- | --- |
| Business problem framing | `README.md`, `reports/project_report.md` | Shows the ability to translate data into a management question. |
| Python data preparation | `scripts/prepare_data.py` | Shows practical data cleaning, feature creation, and export workflow. |
| Dataset sourcing | `scripts/download_dataset.py` | Shows reproducible data acquisition from Kaggle. |
| Relational database design | `sql/01_database_schema.sql` | Shows ability to structure data into customers, orders, products, and sales tables. |
| SQL data validation | `sql/00_load_data_notes.sql`, `sql/02_data_cleaning.sql` | Shows ability to check row counts, quality issues, and reporting readiness. |
| SQL KPI development | `sql/03_kpi_queries.sql` | Shows ability to calculate business metrics using SQL. |
| Sales analysis | `sql/04_sales_analysis.sql` | Shows ability to identify revenue drivers by product, region, market, and time. |
| Profitability analysis | `sql/05_profit_analysis.sql` | Shows ability to find margin leakage and loss-making products. |
| Customer analysis | `sql/06_customer_segment_analysis.sql` | Shows ability to analyze customer segments and customer value. |
| Operations analysis | `sql/07_shipping_analysis.sql` | Shows ability to analyze shipping cost, order priority, and delivery patterns. |
| Time-series analysis | `sql/08_seasonality_analysis.sql` | Shows ability to analyze monthly and quarterly patterns. |
| Power BI dashboarding | `dashboard/power_bi_visuals.pdf` | Shows management-facing dashboard reporting. |

| Documentation | `README.md`, `docs/`, `reports/` | Shows professional communication and portfolio readiness. |

## Interview Talking Points

Use these points when discussing the project in interviews:

1. I did not only analyze sales; I focused on profitable growth and margin leakage.
2. I used Python to prepare the data and export SQL-ready tables.
3. I designed a relational MySQL model with customers, orders, products, and sales.
4. I used SQL to calculate KPIs and investigate sales, profit, discount, customer, shipping, and seasonal patterns.
5. I built a Power BI dashboard to communicate findings to a business audience.
6. I connected findings to business actions such as discount control, regional review, and product-level margin improvement.
