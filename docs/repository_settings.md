# Recommended GitHub Repository Settings

Use this document to set the public-facing repository description and GitHub topics.

## Recommended Repository Description

Paste this into the GitHub repository description field:

```text
End-to-end retail analytics project using Python, MySQL, and Power BI to analyze 51K+ transactions, identify profit leakage, and build executive dashboards for business decision-making.
```

## Recommended GitHub Topics

Add these topics to improve discoverability:

```text
data-analysis
power-bi
mysql
python
business-intelligence
sql
analytics
dashboard
data-visualization
retail-analysis
```

## How To Add Description and Topics

1. Open the GitHub repository page.
2. On the right side, click the gear icon next to **About**.
3. Paste the recommended description.
4. Add the recommended topics one by one.
5. Save changes.

## Why These Topics Matter

| Topic | Reason |
| --- | --- |
| `data-analysis` | Helps position the repo as an analytics project. |
| `power-bi` | Makes the project discoverable for BI dashboard work. |

| `data-visualization` | Shows visual reporting capability. |
