# Power BI Dashboard

This folder stores the exported Power BI dashboard package.

| File | Use |
| --- | --- |
| `power_bi_visuals.pdf` | Exported dashboard pages |

## Dashboard Pages

| Page | Focus |
| --- | --- |
| Executive Overview | Sales, profit, margin, orders, quantity, and trends |
| Sales Analysis | Category, sub-category, region, market, and top product sales |
| Profitability Analysis | Category profit, negative-profit products, profit margin, and discount impact |
| Customer Analysis | Segment sales, segment profit, top customers, and region-segment performance |
| Shipping & Operations | Ship mode cost, order priority, shipping cost by region, and delivery days |
