# Report Assets

This folder contains the business reporting layer for the Global Store Retail Performance Analysis.

| File | Use |
| --- | --- |
| `project_report.md` | Full business report covering data, SQL, Python, Power BI, findings, and actions |
| `executive_summary.md` | Short decision-ready summary |
| `sql_execution_summary.md` | SQL run coverage and Workbench output map |
| `Global_Store_Retail_Analysis_Report.pdf` | Final formatted report export |
| `global_store_analysis_report.pdf` | Exported PDF report from the earlier dashboard/report package |

Use `project_report.md` as the main written report for presentation and GitHub review.
